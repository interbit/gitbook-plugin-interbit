=== Source code

Normal:

[source,js]
----
// A code sample
var check = false
----

With a title:

.Source code with a title
[source,js]
----
// A code sample
var check = false
----

No syntax highlighting:

----
// A code sample
var check = false
----

With callouts:

[source,js]
----
// A code sample <1>
var check = false // <2>
----
<1> The first line, a comment
<2> The second line, actual code
